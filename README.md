# NutriInsightX

NutriInsightX is an AI-powered food label analysis system that extracts and analyzes food packaging information using OCR and machine learning.

## Features

- OCR-based food label extraction
- Ingredient analysis
- Allergen detection
- Food additive detection
- Health risk scoring
- Multilingual label support
- Explainable AI predictions
- Personalized food recommendations

## Dataset

- Open Food Facts
- 283,893+ cleaned food products

## Tech Stack

- Python
- PaddleOCR
- Pandas
- Scikit-Learn
- Transformers
- SHAP
- Streamlit
